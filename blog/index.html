Le tchat

Lors de cette exerçice j'ai du créer un tchat entre deux utilisateurs. Après la création du HTML, j'ai réaliser le JavaScript. Nous devions récupérer les messages de chaque utilisateur et afficher un message indiquant le message de l'utilisateur selon son identifiant. Les messages sont récupérés en utilisant getElementById(monElement).value, puis insérés dans l'élément de discussion en étant enveloppés dans un élément span unique pour chaque message Nous pouvions aussi recupérer et afficher le message de l'utilisateur en utilisant la méthode 'getElementByTagName' qui permet de récupérer tous les éléments HTML qui ont le même nom de tag(meme élément). Cela afficher une alert indiquant le message par rapport à un numéro qui lui été attribuer grace à une variariable 'j' qui a été incrémenté à chaque fois que l'utilisateur a envoyé un message.
